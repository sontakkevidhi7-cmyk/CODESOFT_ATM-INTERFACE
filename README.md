# ATM Interface

## Description
This project is a simple ATM Interface developed in Java. It allows users to perform basic banking operations such as checking account balance, depositing money, and withdrawing money through a menu-driven console application.

## Features
- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-Driven Interface
- Simple and User-Friendly Design

## Technologies Used
- Java
- Scanner Class
- Object-Oriented Programming (OOP)

## Classes Used

### 1. BankAccount Class
Stores and manages the user's account balance.

Methods:
- deposit(amount)
- withdraw(amount)
- checkBalance()

### 2. ATM Class
Provides the ATM menu and handles user interactions.

Methods:
- showMenu()

### 3. ATMInterface Class
Contains the main method and starts the application.

## How to Run

1. Save the file as:
   ATMInterface.java

2. Compile the program:
   javac ATMInterface.java

3. Run the program:
   java ATMInterface

## Project Objective
The objective of this project is to simulate the working of an ATM machine using Java and demonstrate the concepts of classes, objects, methods, loops, conditional statements, and user input handling.
